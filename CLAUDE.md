# cloudflare-workers-resume

A personal resume/portfolio site built with Astro and deployed on Cloudflare Workers. Built to demonstrate Cloudflare platform fluency as part of a job application for the **Forward Deployed Engineer** role at Cloudflare (Toronto, ON).

## Project Goal

Impress the Cloudflare hiring team by hosting a polished resume site *on the platform itself* — showcasing direct, hands-on experience with Workers, the Cloudflare Developer Platform, and modern full-stack tooling.

## Stack

- **Framework**: Astro (SSR via `@astrojs/cloudflare`)
- **Runtime**: Cloudflare Workers (`wrangler.json`, `nodejs_compat` flag)
- **Deployment**: `wrangler deploy`
- **Assets**: Served via Workers Assets binding

## Commands

```bash
npm run dev        # local dev server
npm run build      # astro build
npm run preview    # build + wrangler dev
npm run deploy     # wrangler deploy
npm run check      # build + typecheck + dry-run deploy
```

## Target Job Description — Forward Deployed Engineer, Cloudflare

**Location**: Toronto, ON (Hybrid)

### About the Role

Cloudflare's Forward Deployed Engineers (FDEs) operate at the intersection of product engineering and customer impact. As an FDE, you will be embedded within one of Cloudflare's most strategic global customers, working side-by-side with their engineering teams to build and deploy solutions using Cloudflare's platform. Unlike traditional Solutions Architects or consultants, you will write production code, shape technical architecture, and directly influence how Cloudflare products are used at massive scale.

### Responsibilities

- **Deep Technical Embedding**: Serve as the dedicated, long-term technical partner within a strategic customer's engineering organization.
- **Production-Level Engineering**: Design, build, and deploy production-ready code and configurations directly into customer infrastructure using the Cloudflare platform.
- **Technical Accountability**: Own the account's technical success across the entire Cloudflare portfolio.
- **Operational Integration**: Participate in customer engineering rituals — daily standups, design reviews, real-time incident response.
- **Platform Advocacy & Adoption**: Identify opportunities to accelerate adoption of Cloudflare services across the customer's tech stack.
- **Platform Breadth & Depth**: Deep technical competence across Workers, security, networking, and observability.
- **Feedback Loop & Product Influence**: Surface real-world edge cases and product gaps to Cloudflare's Product and Engineering teams.
- **Strategic Relationship Management**: Trusted technical relationships with Staff+ engineers, Directors, and VPs.
- **On-Site Presence**: Regular on-site presence at customer locations.

### Requirements

- 5+ years professional software engineering experience; builder track record with production-grade systems.
- Production ownership of mission-critical services — on-call, incident response, "five nines" uptime.
- Active practitioner — ship-first mentality, high velocity, current with modern frameworks and deployment patterns.
- Full-stack depth: frontend performance, backend logic, database optimization, distributed infrastructure.
- AI-native development — integrated AI-augmented workflows (e.g., Windsurf, OpenCode) into daily dev cycle.
- Systems design & strategic thinking — decompose complex business requirements into scalable, resilient architectures.
- Cloud ecosystem fluency — deep experience with at least one major cloud provider (AWS, GCP, Azure).
- High agency — self-starter who thrives in "zero-to-one" environments.
- Executive presence — comfortable in high-stakes discussions with VP-level stakeholders.

### Nice to Have

- Direct experience building on Cloudflare Workers or the broader Cloudflare Developer Platform.
- Previous experience at a fast-paced startup or in a customer-facing engineering role operating within a partner's codebase.
- Visible public profile: open-source contributions, technical blogging, or conference speaking.
